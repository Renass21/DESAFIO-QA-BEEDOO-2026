# DESAFIO-QA-BEEDOO-2026


**Qual você acredita ser o objetivo da aplicação?**

O objetivo da aplicação deve ser um painel de administração de cadastrar cursos, editar, excluir e listar os cursos.
Mostrando a quantidade de vagas, periodo, nome do professor, descrição e link do curso 

**Quais são os principais fluxos disponíveis?**

Cadastrar e listar os cursos ofertados.

**Quais pontos do sistema você considera mais críticos para teste?**

Na minha visão por ser um painel de adm dos cursos, primeiro a autenticação, por segundo o formulário de cadastro de cursos focando na funcionalidade do sistema.
